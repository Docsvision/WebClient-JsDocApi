Порядок сборки:
1) Выкачать последнюю версию ВК, пересобрать сервер, собрать клиент ВК командой npm run build:prod
2) Скопировать из ВК папку Bin/TypeScriptDefinitions в данную папку, удалив предвариетльно то что здесь было
3) Проверить содержимое README.md в родительской папке (обновить версию, добавить ссылки на предыдущую версию).
4) Выполнить в командной строке npm i typedoc@0.11.1 -g -D handlebars@4.5.3
5) Обновиь generate.bat - поднять версию в параметре --name команды запуска typedoc.
6) Запустить generate.bat
7) Запустить generateDocsMap.bat
8) Открыть Git Bash в текущей папке и выполнить node ./build-index.js
9) Закоммитить все
