Class AdaptiveMenuBar

internal

Представляет собой адаптивное меню.

Пример использования:

<AdaptiveMenuBar expanded={this.state.menuBarExpanded} >
    <AdaptiveMenuContent>
        <AdaptiveMenuItem onClick={() => console.info("Command 1 clicked") } >
            Комманда 1
        </AdaptiveMenuBarItem>
        <AdaptiveMenuBarItem onClick={() => console.info("Command 2 clicked")} >
            Комманда 2
        </AdaptiveMenuBarItem>
    </AdaptiveMenuContent>
</AdaptiveMenuBar>
