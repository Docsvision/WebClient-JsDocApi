    Показывает модальное окно с запросом подтвердить какое-либо действие.

    Parameters

    • content: React.ReactNode

      Текст запроса

    • Optional customCaptionText: React.ReactNode

      Заголовок диалога

    • Optional customButtonOkText: React.ReactNode
    • Optional customButtonCancelText: React.ReactNode
    • Optional hideByBackdropClick: boolean

    Returns Promise<any>

    Promise объект, который перейдет в состояние "resolved" при согласии, и "rejected" при отказе.

    Пример использования:

    MessageBox.ShowConfirmation("Вы действительно хотите удалить файл?").then(() => { this.removeFile(); }).catch(() => { // Пользователь отменил удаление });
