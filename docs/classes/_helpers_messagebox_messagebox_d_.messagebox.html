    • Defined in Helpers/MessageBox/MessageBox.d.ts:71

    Показывает модальное окно с запросом подтвердить какое-либо действие.

    Parameters

    • content: React.ReactNode

      Текст запроса

    • Optional customCaptionText: React.ReactNode

      Заголовок диалога

    Returns JQueryDeferred<any>

    JQueryDeferred объект, который перейдет в состояние "resolved" при согласии, и "rejected" при отказе.

    Пример использования:

    MessageBox.ShowConfirmation("Вы действительно хотите удалить файл?").done(() => { this.removeFile(); }).fail(() => { // Пользователь отменил удаление });
