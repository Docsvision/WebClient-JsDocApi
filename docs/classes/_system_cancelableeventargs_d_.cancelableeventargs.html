Class CancelableEventArgs<T>

Представляет собой событие для действий, подписчики которого могут его отменить. Заметка: подписчики события могут выполнять долговременные асинхронные операции перед продолжением или отменой действий событий. Пример использования:

this.cardOpening = CancelableEvent.Create(this.state.wrapper); ... this.cardOpening.trigger({ cardModel: model }) .canceled(this.back) .accepted(() => callback(model));

Заметка: подписчики события должны использовать только интерфейс ICancelableEventArgs публичного API, поскольку интерфейс CancelableEventArgs только для внутреннего использования инициаторов событий.
