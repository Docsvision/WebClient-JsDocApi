Protected setParamValue

  • setParamValue(propName: string, newVal: any, initial: boolean): void

  • Inherited from BaseControlImpl.setParamValue

    • Defined in layout-extension.d.ts:2765

    Обработчик, вызываемый всякий раз, когда установливается значение параметра. Происходить это может в следующих случаях:

    1. При инициализации компонента (из метода componentWillMount).
    2. При получении новых props компонента (из метода componentWillReceiveProps). Как правило, новые свойства передаются интерфейсным компонентом при вызове BaseControl.setParamValue.

    Метод Реализует следующую логику:

    1. Если объявлено set-свойство с декоратором handler, то возвращается значение данного свойства;
    2. Иначе устаналивается значение в state.

    Parameters

    • propName: string
    • newVal: any
    • initial: boolean

      Значение истино, если метод вызывается при инициализации компонента (из componentWillMount).

    Returns void
