Optional searchQuery

searchQuery: string
  • Defined in layout-extension.d.ts:498

Search query, entered by user. Result should contain tree, enough to show n-th matched element, where n is searchResultNumber. So, result shoudl contain element itself, and all its parent. It also should contain root elements (without parent), to keep tree pretty when navigate between results.
