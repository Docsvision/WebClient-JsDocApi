Optional iconClass

iconClass: string
  • Defined in layout-extension.d.ts:780

Show specified icon to the left of the text. Class should define width and heihgt background, representing an icon Preffered icon width is 18px (it is loading icon width), but you can use icon of any size. If icon is not 18x18, then you should setup loadingIconClass, and specify your loading icon, otherwise text will jump on loading on and off. Default value: null
